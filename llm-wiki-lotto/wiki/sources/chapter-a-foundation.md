# 챕터 A — 로또 분석의 해법

cryingbird의 이론적 기초 3부작 + 확률 통계론 개요.

## 1부: 베르누이 시행

출처: `raw/lotto/220367853909_...1.-로또는-베르누이-시행이다..md`

- 매 회차는 **독립 시행** — 이전 회차 결과가 다음에 영향 없음
- 모든 번호 출현 확률은 매 회차 동일 (1/45)
- 따라서 **예측 불가·불규칙** → [[concepts/bernoulli-and-lln|선형 공식·회귀분석 부적합]]
- 대다수 로또 분석이 선형 공식을 쓰는 것은 이론과 모순

→ 상세: [[concepts/bernoulli-and-lln]]

## 2부: 대수의 법칙

출처: `raw/lotto/220367872258_...2-.-로또는-대수의-법칙을-따른다..md`

- 장기적으로 모든 번호 출현 횟수는 **수렴** (평균 1/45)
- 단기 이격은 있으나 장기적으로 균등화
- 분석은 단기 이격을 활용하되, 장기 균형을 염두

## 3부: 해법 제시

출처: `raw/lotto/220367883675_...3.-로또-분석의-해법-제시.md`

로또 3대 특성 정리:

1. 베르누이 시행 → 예측 불가, 불규칙, 단기 **이격**
2. 단기 이격 → 빈도 0~N으로 넓게 분산 (10게임·30게임 표본)
3. 장기 대수의 법칙 → 출현 횟수 균등화

**30게임**의 의미: 모든 수가 최소 1회 출현하는 분기점, 이격 분포가 [[concepts/frequency-deviation|가우스 정규분포]] 형태를 띠기 시작.

**해법**: 단기 이격(빈도 분산)을 읽어 조합을 좁히되, 베르누이 독립성을 잊지 않을 것.
