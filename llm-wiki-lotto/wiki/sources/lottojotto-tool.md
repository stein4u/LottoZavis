# 로또조또 도구 문서

## 개요

[[concepts/lottojotto|로또조또]]는 cryingbird가 5~7년간의 로또 구매·분석 경험을 바탕으로 개발한 **로또 분석 전용 프로그램**(로또번호추출분석기). 출처: `raw/lotto/220184385504_로또분석전용프로그램-로또조또의-개발.md`

개발 동기: 기존 앱 수준의 랜덤 생성기에 불만, 통계·확률 기반 체계적 분석·조합 추출 필요.

## 설계 핵심 (확률 통계론)

출처: `raw/lotto/220187252131_로또분석-방법--로또조또에-적용된-로또-확률-통계론.md`

- **대수의 법칙(베르누이)** + 구간별·숫자별 빈도 함수화
