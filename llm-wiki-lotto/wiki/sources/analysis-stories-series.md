# 로또 분석 이야기 시리즈 (1~37)

cryingbird가 확률론 기반으로 직접 집필한 연재 시리즈. 2015.4 ~ 2016.8.

## 시리즈 구조

| 구간 | 화수 | 주제 |
|------|------|------|
| 기초·비판 | 1~4 | 제외수, 업체 비판, 고정수 없음, 분석 방법론 |
| 외국 분석법 | 5~8 | 후나츠 사카이, 게일 하워드, 9궁도, 포지션 |
| 실전·정복 | 9~14 | 실전 활용, 효용성, 완전 정복 1~3 |
| 예상수 추리 | 15~17 | 예상수 추리기 1~3 |
| 이론 심화 | 18~29 | 조작 논쟁, 선형 비판, 조합론, 필터링, 빈도 이격·행보 |
| 숲과 나무 | 30~37 | 거시적 분석, 지표 활용, 재출현 간격, 한계 이격 |

## 이론과 긴장하는 화

> [!warning] 4화 — 1등이 보인다
> 제목은 1등 당첨을 암시하나, [[concepts/bernoulli-and-lln|베르누이 시행]]에 따르면 매 회차 1등 확률은 불변. 실질 내용은 **분석 프레임·조합 필터링**으로 해석해야 함.

> [!warning] 29화 — 유력 1조합은 정해져 있다
> "정해진 1조합"은 [[concepts/combination-over-numbers|조합 단위]]에서 필터·이격 기준으로 **가장 납득 가능한 조합**을 고른다는 뜻으로 읽어야 함. 예측이 아닌 **주간 최적 후보 선정**.

## 시리즈 핵심 메시지

1. **제외수**로 조합 공간을 줄이는 것이 확률적으로 가장 효과적 (`1`화)
2. **선형·회귀 분석은 부적합** — 로또는 불규칙 시행 (`19`화, [[concepts/bernoulli-and-lln]])
3. **조합 단위**로 접근 — 개별 번호 고정 금지 (`20`화) — [[concepts/clustering-dispersion]]로 분산
4. **빈도 이격·행보**로 단기 추세 읽기 (`24`~`25`, `28`, `33`~`35`화) — [[concepts/limit-deviation]], [[concepts/reappearance-gap]]
5. **필터링**으로 표본 조합에서 통계 추출 (`23`화)
6. **숲(전체 분포)**을 보고 **나무(개별 수)**에 집착하지 말 것 (`30`~`31`화)

> [!warning] 비판적 해석
> 시리즈는 분석으로 당첨 확률을 높일 수 있다는 전제를 가지나, [[concepts/bernoulli-and-lln|베르누이 독립성]]에 따르면 매 회차 당첨 확률 자체는 변하지 않는다. 위키는 방법론 기록과 비판적 맥락을 함께 유지한다.
