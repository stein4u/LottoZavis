# 제외수

## 정의

6개 조합 번호를 만들 때 **의도적으로 빼는 번호**. 출처: `raw/lotto/220325344697_...1.---제외수의-진실.md` — [[sources/analysis-stories-series]] 1화.

## 조건부 확률 (위키 해석)

[[concepts/bernoulli-and-lln|베르누이 전제]]와 저자의 확률 표를 함께 읽어야 함.

| 구분 | 의미 |
|------|------|
| **무조건부** | 매 회차 6개 번호 조합의 1등 확률 = **1/8,145,060** (변하지 않음) |
| **조건부** | 제외수 N개가 **모두 적중**(당첨 6개에 포함 안 됨)했을 때, 남은 조합 공간에서의 확률 |

아래 표의 1/382만·1/162만·1/59만은 **"제외가 맞았다는 조건 하에"** 남은 번호만으로 조합을 만들 때의 확률. 제외가 틀리면 조건 불성립.

```
P(1등) ≠ P(1등 | 제외 적중) × P(제외 적중) + ...
```

실질 효과: 제외수로 **탐색 공간을 줄이는 것**. 1등 확률 자체를 바꾸지는 않음.

## 확률 효과 (저자 주장, 조건부)

제외수가 **적중**(제외한 번호가 실제 당첨 6개에 없음)하면 조합 공간이 줄어듦:

| 제외수 적중 개수 | 1등 확률 (저자 표기) |
|------------------|----------------------|
| 5개 성공 | 1/382만 |
| 10개 성공 | 1/162만 |
| 15개 성공 | 1/59만 |

(기본 45C6 = 1/8,145,060 대비)

## 1화 실험

1~10을 제외수로 가정, 랜덤 10조합 시뮬레이션 → 평균 제외수 적중률 약 **87%** (10개 중 0~2개만 당첨번호에 포함).
