# 몰림·쏠림·분산

cryingbird 분석법의 **핵심 원리**. 출처: `raw/lotto/220368709349` (분석 이야기 22화)

## 정의

| 용어 | 의미 |
|------|------|
| **몰림** | 특정 빈도 구간에 번호가 집중 출현. 대수의 법칙이 이격을 메우려는 현상 |
| **쏠림** | 특정 지표·구간으로 치우침. 정상 이격에서 벗어난 편중 |
| **분산** | 조합 구성 시 빈도 구간(0·1·2빈도 등)을 **고르게** 배치 |

## 핵심 원리 (22화)

1. 30게임 기준 평균 빈도 4.66 주위로 **가우스 분포** 형태이나, 로또는 완전 정규분포가 아님
2. 단기 **쏠림** 발생 → 역이용하여 분산 비율 결정
3. 조합 시 **0빈도·1빈도·2빈도**를 뚜렷히 분산 배치해야 함
4. 몰림에도 **한계** 있음 — 최근 10게임에 6~7회 출현은 없음
5. 구간(번대)별로도 비중만큼 **분산** — 한 구간에 몰아치지 않음

## 실전 적용

- 0빈도 구간 7수 중 3수만 포함 → 분산 취약, 제외
- 1빈도 구간 15수 중 4수 포함 → 분산 취약, 제외
- 쏠림이 강할 때는 해당 구간 **집중 공략** 조합 비중 확대 (단기적)
- [[concepts/filtering|필터링]]·[[concepts/frequency-deviation|빈도 이격]]과 연동
- 1231회 실측 몰림: 27(8회), 28·31(7회) → [[analyses/freq-trend-2026-07-05]]
